# POSE course readings index

This directory contains readings and resources for the POSE (Pathways to Open Source Ecosystems) course.

## Required books

1. **Talking to Humans** - Giff Constable and Frank Rimalovski
2. **Value Proposition Design** - Greg Bernard, Yves Pigneur, Alexander Osterwalder and Alan Smith
3. **The Business of Belonging** - David Spinks
