# Calendar scheduling setup

## Option 1: Cal.com (recommended - free, open source)

1. Go to https://cal.com
2. Create team account "PolicyEngine"
3. Create event type:
   - **Name:** POSE Ecosystem Interview
   - **Duration:** 30 minutes
   - **Description:** "Thanks for chatting with us! We're exploring how to build sustainable open-source policy tools and want to learn from your experience."
   - **Location:** Google Meet (auto-generated)
4. Set availability:
   - Connect Google Calendars for Max, Pavel, Dan
   - Set overlapping availability windows
5. Share link: `cal.com/policyengine/pose-interview`

## Option 2: Calendly

1. Create Calendly account
2. Create 30-min event type
3. Connect team calendars
4. Share link: `calendly.com/policyengine/pose-interview`

## Google Calendar alternative

If not using scheduling tool:
1. Create "POSE Interviews" calendar
2. Share with team
3. Block interview slots manually
4. Send calendar invites directly

## Interview logistics

- **Platform:** Google Meet (everyone has it)
- **Recording:** Ask permission, use Google Meet recording or Otter.ai
- **Notes:** Take in real-time, add to POSE Tracker within 24 hours
- **Duration:** 30 min scheduled, may run 45 min

## Timezone note

Most interviews will be US Eastern or Pacific time.
UK interviews: schedule 9am-12pm EST (2pm-5pm GMT).
